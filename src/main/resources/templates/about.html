About TaxiBook System

TaxiBook is a complete Spring Boot major project for online taxi booking, ride sharing, driver assignment, shortest route calculation, admin analytics, and service management.

Book Now Find Shortest Path

Backend Connected

All booking, contact, ride sharing, service and admin data is stored through Spring Data JPA and MySQL.

Smart Driver Assignment

When a customer books a taxi or ride share, the backend assigns a driver and vehicle automatically.

Shortest Path Engine

The shortest route module uses Dijkstra algorithm to show distance, time, route steps and map view.

Designed for a strong project demo

This project shows real MVC flow: Thymeleaf frontend, Spring controllers, service layer business logic, JPA repositories, MySQL database, Spring Security admin login and live dashboard analytics.
